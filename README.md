### a Java Library for testing Spigot Plugin 

## Development / Build
1. ```shell
   git clone https://github.com/Relluem94s/RelluSpigotMockito.git
   ```

2. ```shell
   git checkout $(git describe --tags $(git rev-list --tags --max-count=1))
   ```

3. ```shell
   mvn clean install
   ```
